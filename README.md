# OfficeTools

OfficeTools is a local-first Electron desktop application for UOS ARM64 office workflows.

Phase 1 focuses on the desktop shell, Excel split, and Excel merge workflows.

## Development

This repository uses pnpm workspaces.

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm lint
```
